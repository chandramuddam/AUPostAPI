Description:

This project is a small and simple example for getting and validating the responses from Australia post api
for various scenarios on different end points 

End points covered for Australia post API:

1. Retrieve the list of service codes
2. Retrieving the list of domestic letter thickness values
3. Retrieve the international parcel services for a given country
4. Retrieve the post code for a given suburb and state
5. Calculate the postage cost

Implementation details:

Used Rest assured using Java, Cucumber, Log 4j and JUnit 

Usage:

Run using Maven command - mvn test
